## 本次后端修改

### Dashboard API 增强
- `GET /api/stats/dashboard` 新增 `trend`（近7天趋势）和 `top_rooms`（热门 Top 5）字段
- 新增 `buildTrendData()` 和 `buildHotRooms()` 辅助方法

### 企微日程同步 API
- `GET /api/wecom/calendar/events` — 获取已同步日程列表
- `POST /api/wecom/calendar/sync` — 手动触发全部同步
- `POST /api/wecom/calendar/sync/:eventId` — 同步单个日程

### Excel 导入导出
- `POST /api/rooms/import` — 真实 Excel 解析（名称/楼层/楼栋/容量/类型）
- `GET /api/rooms/export` — 真实 Excel 生成（含样式、列宽）
- `GET /api/stats/export` — 统计报表 Excel 导出
- 使用 `excelize/v2` 库
